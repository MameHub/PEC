Escribe un programa que ponga en práctica la siguiente lógica: a) Mostrar el siguiente menú y el usuario deberá elegir una de las opciones. 1.- Área de triángulo 2.- Área de rectángulo 3.- Media aritmética de dos números 4.- Salir b) En función de la opción que elija el usuario se le pedirán los datos necesarios en cada caso. c) El programa calculará el resultado y lo mostrará en la consola. d) La lógica anterior se repetirá indefinidamente hasta que el usuario pulse la opción 4 para salir que mostrará un mensaje por consola, "Fin del ejercicio". e) Si la opción no esta entre el rango 1-4 se mostrará un error mediante la función alert() y se volverá a pedir otra opción.
